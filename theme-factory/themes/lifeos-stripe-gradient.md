# Life-OS · Stripe Gradient（明亮專業漸層系）

白底深海軍藍文字＋紫粉漸層彩帶，乾淨、可信、帶一抹未來感。適合簡報、提案、對外文件。

> 來源：stripe.com 官網 CSS 直抓（2026-07-05，`--gradient-color` 等變數原值，**中高信心**——漸層色可信，中性色取樣自部分 CSS 檔）。

## Color Palette

- **Background**: `#ffffff` - 主背景
- **Surface Tint**: `#e5edf5` - 淺藍灰區塊底
- **Text Navy**: `#2d2d44` / `#424770` - 主文字（深海軍藍，Stripe 的招牌「不是黑的黑」）
- **Text Muted**: `#929eaa` - 次要文字
- **Gradient Purple**: `#533afd` - 漸層主錨點
- **Gradient Pink**: `#fb76fa` → `#ec8fff` → `#da56ed` - 漸層過渡帶
- 點綴：橘 `#ff6118`、桃紅 `#ea2261`、金 `#f5a623`（icon/小標籤用）

## Typography

- 原版：Söhne（sohne-var）＋ Source Code Pro
- **本機替代**：Work Sans（canvas-fonts 自帶）或 Outfit、等寬 Red Hat Mono
- **繁中搭配**：Noto Sans TC（字重對比拉大：Bold 標題／Light 內文）

## Spacing / Feel

大量留白、區塊間距大、漸層只出現在 1-2 處斜切彩帶或標題文字（絕不滿版鋪）；圓角中等；按鈕實心藍紫。
Feel：專業、可信、乾淨、國際、輕未來感。

## Best Used For

pptx 提案簡報、對外報告、商用版 Life-OS 文件、landing page、有「說服」任務的輸出。

## Anti-patterns

漸層不要超過兩處（AI slop 重災區）、不要彩虹漸層、內文不要用純黑（用海軍藍）。
