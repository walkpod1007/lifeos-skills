# Life-OS · Anthropic Editorial（暖調編輯系）

溫暖的象牙底＋陶土橘點綴，書卷氣、沉穩、像一本好書的內頁。適合長文、雜誌 PDF、閱讀型輸出。

> 來源：anthropic.com 官網 CSS 直抓（2026-07-05，`--swatch--*` 變數原值，**高信心**）。

## Color Palette

- **Ivory Light**: `#faf9f5` - 主背景
- **Ivory Medium**: `#f0eee6` - 次背景/卡片
- **Ivory Dark**: `#e8e6dc` - hover/分隔
- **Slate Dark**: `#141413` - 主文字（近黑）
- **Slate Light**: `#5e5d59` - 次要文字
- **Clay**: `#d97757` - 主強調色（陶土橘）
- **Accent**: `#c6613f` - 強調色深階
- 擴充色（少量點綴）：Olive `#788c5d`、Sky `#6a9bcc`、Kraft `#d4a27f`、Fig `#c46686`

## Typography

- 原版：Anthropic Serif／Tiempos Text（襯線）＋ Anthropic Sans ＋ JetBrains Mono
- **本機替代**（canvas-design/canvas-fonts 自帶）：標題 Lora Bold 或 Crimson Pro、內文 Lora、等寬 JetBrains Mono
- **繁中搭配**：Noto Serif TC（標題/內文）＋ PingFang TC（UI 短字）

## Spacing / Feel

留白極多、單欄窄版心（閱讀寬度）、圓角小而克制、無陰影或極淡。
Feel：書卷、溫暖、沉穩、信任、編輯部。

## Best Used For

magazine-doc 雜誌 PDF、長文報告、gdoc-article、閱讀型 dashboard、書單/片單清單。

## Anti-patterns

不要漸層、不要霓虹色、不要深色模式硬翻（這套的靈魂就是暖白紙感）。
